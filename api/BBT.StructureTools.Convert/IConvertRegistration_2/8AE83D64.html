Summary

Registers a to many relationships on source and corresponding target. TSourceValues are filtered with sourceFunc.

    where TSourceValue : class where TTargetValue : class where TConcreteTargetValue : TTargetValue, new() where TReverseRelation : class where TConvertIntention : IBaseConvertIntention

Type Parameters

Name Description
TSourceValue The type of the ist entries on TSource.
TTargetValue The type of the ist entries on TTarget. The TTarget type and the TTargetValue value must be of type ICollection<T>.
TConcreteTargetValue The concrete implementation of TTarget.
TReverseRelation The reverse relation of the created target value. Must be a base type of TTarget.
TConvertIntention The intention of the conversion.
